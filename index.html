HTML.js

A DOM implementation and a html parser

Download this project as a .zip file Download this project as a tar.gz file

A basic DOM implementation that includes a HTML parser and a Plugin API

Just require it require('html')

Then create a Document new html.Document('You html')

That just has a DOM API

You can document.createElement('div' || 'you tag name')

To use plugins you just create a document passing in another argument: what plugins you want loaded.

And to write your own plugins:

html.plugins.add('plugin name', function(plugin) {
     // You plugin here
     // You can register tags
     plugin.tags.add('tag name', { // Tag Spec
          endTag: Boolean, // do we need an end tag ( if false do not even look for one )
          html: Boolean, // can we have html inside of this tag
          comments: Boolean, // can we have comments inside of this tag
          create: Function, // this is run on any new elements
          prototype: Object // this is applied as a prototype on new elements
      });

      // or new ways to add integrate
      plugins.activators.add('type', {
          activate: Function, // activate a plugin
          deactivate: Function
      });
});
